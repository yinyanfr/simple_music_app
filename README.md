### 1. Get Started

**NodeJS v8+ & MongoDB 3.6+**

```bash
$ service mongod start
$ cd api
$ npm i
```

**Test**

```bash
$ npm test
```

**Run**

```bash
$ npm start
```

then http://localhost:30706

### 2. List of Technologies

- Server
  - NodeJS
  - MongoDB
- Web
  - React
- Style
  - Bulma
